# Welcome to the PlaceMark 1.0 App  

## Overview  

The PlaceMark 1.0 App, is a Point-of-Interest (PoI) manager, which allows users to add as many PoI they want.

The App will be developed with Waterfalls PoI in mind. The idea is that users can placemark waterfall locations and include additional details such as "transportation", "restaurants", "accommodation", etc. 

The PoI can be grouped using tags like "easy, okay, hard to get there" and user-defined collections like "waterfalls to go, visited, reviewed", and facilities like "has transport, accommodation, restaurants".

The App will have a feature that allows others to access the shared POI. The user who created the PoI will be marked as "admin" and will be able to add others to access it, which will be marked as "invited". The admin user can view all invited users in the admin dashboard and revoke permissions for any of them.
